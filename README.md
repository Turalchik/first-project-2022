### <a name="Empasis"></a>Выделение элементов
Текст выделен **полужирным**. И этот текст тоже будет __полужирным__.
Текст выделен *курсивом*. И этот текст тоже будет _курсивным_.

***

Текст выделен ***полужирным курсивом***
